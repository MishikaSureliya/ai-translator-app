## üöÄ Installation & Local Development

### 1. Requirements
Ensure you have Python 3.9+ installed on your computer.

### 2. Setup
Clone this repository or download the files.

Open your terminal in the project folder and run:

```bash
pip install -r requirements.txt
```

### 3. Run the App
Launch the Streamlit web server locally using:

```bash
streamlit run app.py
```

The application will automatically open in your default web browser at `http://localhost:8501`.

This repository is strictly structured and optimized for instantaneous Free Deployment via Streamlit Community Cloud.

### Deployment Steps:
1. Upload your code to a public or private GitHub repository.
2. Ensure `app.py`, `style.css`, and `requirements.txt` are together in your repository's root directory.
3. Sign up or log into [Streamlit Community Cloud](https://share.streamlit.io/).
4. Click **New app**.
5. Select your designated GitHub repository from the dropdown. 
6. Set the Main file path to `app.py`.
7. Click **Deploy!** 

Streamlit's servers will instantly install the required packages from your `requirements.txt` file and spin up a live, public, shareable URL to your beautiful AI translator app!
